Deep Learning CV and NLP Application

This application started as a single SQL file that executed a Competitor Analysis. Because Amazon Redshift does not support variable declaration, consultants had to manually run the code 5-10 times and copy paste the results for each analysis. This cost them around 30 minutes. I started by turning the SQL file into a Python script that automatically ran the analysis for every ID and concatenated the results into a single file. Next, I built various interfaces like a CLI and a GUI with the Tkinter library. Finally, I settled on the use of the Kivy library. Therefore, the application uses a combination of Python, SQL and Kivy code. Later, I added multiple analyses to the application like a Study ID Collector, a 'prequery' before the Competitor Analysis and a Study Information Collector.
