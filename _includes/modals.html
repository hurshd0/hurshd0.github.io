Memefly

Deep Learning App that generates Dank Memes

For millenniums, artists and writers have been able to generate captions from images or from images to paintings. It goes as far back to ancient Egyptian tomb paintings from 2000 B.C., to classic Chinese paintings from 1000 A.D. to the popular memes of the 2010s.

For Memefly, we attempted to generate memes from image inputs. We trained a GRU network par-injected with image embedding extracted using pre-trained InceptionV3 and sentence embedding using methods outlined in Show and Tell: A Neural Image Caption Generator [Vinyals ‚Äò14], Where to put the Image in an Image Caption Generator [Tanti ‚Äò17], and Dank Learning: Generating Memes Using Deep Neural Networks[Peirson ‚Äò18]. We also implemented beam-search as suggested by Professor Andrew Ng. We scrapped from various online resources a small dataset of around 108 memes and 20,000 descriptions for our training data. We implemented our network using Tensorflow 2.0, and used offline learning to generate captions and store them in Cloud MongoDB database.

Spotifier

Song Recommender App

Why Spotifier?

One of my passion is discovering new music, whether its new genres or artists, more often recommendations come from friends or through sub-reddits or blogs. What was missing was an app that recommends tracks based on their similarity not similar artist based on genres.

How do you approach this problem?

  1. First, given the problem where we want similar tracks instead of similar artists which points to looking at audio features of each individual tracks so we decided to use readily available Kaggle's Spotify Audio Features Dataset.
  2. Second, once we have dataset we want to perform ETL task and load finalized data to a AWS RDS PostgresSQL database.
  3. Third, we want to run our choosen Machine Learning algorithm, since we don't have user dataset or their preference we can't create Collaborative Filtering models, which leaves us with Unsupervised Nearest Neighbors, Clustering or Neural Networks. Starting with Unsupervised Nearest Neighbors techniques like - kNN, Ball Tree, KDTree, in this case would be a great start since the goal is to create fastest prototype.

How does it work?

When you search for a song it queries the database finds the track you are looking for along with its Top 10 recommendations.

nCoV19.US

COVID-19 Tracker

nCoV19.us is a U.S. specific COVID-19 tracker site that provides up to date information on the spread of coronavirus.

Features include:

  • Tested, confirmed, deaths data for state and county level
  • Testing center locations in the US
  • Dropdown to filter data by state
  • Trajectory visualizations to compare states
  • Twitter/News feed
  • SMS updates
